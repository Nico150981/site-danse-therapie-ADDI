Pour ajouter vos propres images :
1. Glissez-déposez le fichier ici (interface web GitHub, ou en local).
2. Dans la Google Sheet, indiquez le chemin dans la colonne "image",
   par exemple : /images/mon-portrait.jpg
3. Le site les optimise automatiquement (WebP, plusieurs tailles, chargement
   différé) grâce au module @11ty/eleventy-img - vous n'avez rien d'autre à faire.

Formats acceptés : jpg, jpeg, png, webp.
